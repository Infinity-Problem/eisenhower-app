EISENHOWERPriority matrix for getting actual work done

A standalone Windows desktop app where the Eisenhower decision matrix is the entire UI. Local-first storage, time tracking per task, drag-and-drop between quadrants, and optional one-way sync to Google Calendar.

What it does

  • Add tasks straight into any of the four quadrants, or use a global input column with a date picker.
  • Drag tasks between quadrants when priorities shift.
  • Adopt a task to start a live time tracker; the elapsed time accrues to that task across sessions.
  • Filter the board by Today / Tomorrow / This Week / This Month / Past Due / Done.
  • Battery shows completion progress for the active filter; donut charts split open tasks and tracked time across the four quadrants.
  • Optional Google Calendar sync mirrors any task with a due date as an all-day event.

Privacy

All task data lives on your machine in %APPDATA%\Eisenhower\tasks.json. Google Calendar sync, when enabled, only pushes tasks-with-dates to a calendar you choose. See the Privacy Policy for details.
